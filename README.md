
# Shipment API

This is a simple RESTful API for managing shipments.

### Endpoints

- `GET /api/shipments`: Get all shipments
- `GET /api/shipments/:id`: Get a single shipment
- `POST /api/shipments`: Create a shipment
- `PUT /api/shipments/:id`: Update a shipment
- `DELETE /api/shipments/:id`: Delete a shipment

### Swagger Docs

- Visit `https://your-app-domain/api-docs` to access the Swagger API documentation.

### Setup

1. Install dependencies:
   ```
   yarn add
   ```

2. Start the server:
   ```
   yarn start
   ```

3. Test the API using Postman or curl or you also directly access the swagger documentation.
